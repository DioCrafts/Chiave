# Documentación de la API

## Endpoints

### POST /login
- **Descripción**: Iniciar sesión.
- **Parámetros**:
  - `username`: nombre de usuario.
  - `password`: contraseña del usuario.

### POST /register
- **Descripción**: Registrarse como nuevo usuario.
- **Parámetros**:
  - `username`: nombre de usuario.
  - `email`: correo electrónico.
  - `password`: contraseña del usuario.
